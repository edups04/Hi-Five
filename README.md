**To run the frontend server:**
  * go to terminal and cd frontend
  * in the terminal type "**npm run dev**"

**To run the backend server:**
  * in a new terminal and cd backend/server
  * in the terminal type "**node index.js**"

**To run the machine recognition:**
  * in a new terminal and cd Hi-Five_Recognition/ml
  * in the terminal type "**python app.py**"

----------------------------------------------------------------------------------------------------------

**Libraries to install:**
  * go to terminal and cd frontend
  * in the terminal type **npm install bootstrap axios react-router-dom**
  * go to terminal and cd backend/server
  * in the terminal type **npm install express mongoose cors nodemon bcrypt express dotenv multer**
  * in the terminal go to cd/Hi-Five_Recognition/ml and type **pip install -r requirements.txt**

----------------------------------------------------------------------------------------------------------

**Working Features (System is Connected to a cloud database (MongoDB))**
 * Log in and Sign up, Including Google Log in
 * Forgot Password
 * Update Password in Profile/Settings
 * ASL Sign Language Recognition (Record, Save and Export)
